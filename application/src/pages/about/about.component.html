Description

This application serves as a web-based control and monitoring interface for the fireCNC hardware. It provides real-time data visualization, configuration management, and system diagnostics. The front-end is built with Angular in zoneless mode, leveraging signals for state management to create a highly reactive and performant user experience.

Technology Stack

  • Framework: Angular (Zoneless)
  • State Management: Angular Signals
  • Styling: Tailwind CSS
  • Communication: WebSocket for real-time data
  • Persistence: Browser localStorage
