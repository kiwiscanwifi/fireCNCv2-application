Module Notes

The RS485 TO ETH (B) is an industrial rail-mount RS485 to Ethernet serial server. It is used in the fireCNC system to provide a reliable communication bridge between the main controller and devices on the RS485 bus, such as servo motors.

Key Features

  • Supports TCP server, TCP client, UDP mode, and UDP multicast.
  • Rail-mount casing design, easy to install.
  • Power supply range of 5 ~ 36V DC.
  • Built-in web page for configuration.
  • Provides Modbus gateway support, suitable for Modbus TCP to Modbus RTU.
